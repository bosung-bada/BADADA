# 🌊 BADADA

> 함께 들어가고, 함께 나온다.

BADADA는 해루질, 프리다이빙, 갯벌체험, 낚시를 위한
실시간 위치 공유 및 안전 지원 플랫폼입니다.

---

# 🌊 프로젝트 목표

누구나 안심하고 바다를 즐길 수 있도록
실시간 위치 공유와 SOS 기능을 제공합니다.

---

# 🎯 핵심 기능

- ✅ 회원가입 / 로그인
- ✅ 친구 추가
- ✅ 팀 생성
- ✅ 팀 위치 공유
- 🚧 실시간 지도
- 🚧 SOS
- 🚧 자동 위험 감지
- 🚧 물때 / 날씨
- 🚧 해루질 기록

---

# 🌊 BADADA의 흔들리지 않는 원칙

1. 바다에서는 버튼 하나면 충분하다.
2. 사람의 생명과 안전이 모든 기능보다 우선이다.
3. 복잡한 기능보다 직관적인 경험을 만든다.
4. 실제 해루질하는 사람이 매번 사용할 기능만 만든다.
5. 함께 들어가고, 함께 나온다.
